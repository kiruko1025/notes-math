# Completeness
A set of logical connectives is complete if every truth function can be expressed using only those connectives.
$\{ \land, \lor , \neg\}$ is complete.


# CNF
ands of ors of literals

Any wff can be converted to CNF.

# DNF
ors of ands of literals

Any wff can be converted to DNF. (but it may be exponentially larger)
